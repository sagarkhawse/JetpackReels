# JetpackReels
JetpackReels app is kind of instagram reel clone made using Jetpack Compose.

Technologies used to create this project are as follows :- 
- Jetpack Compose
- Kotlin
- Mvvm with Clean architecture
- Retrofit
- Exoplayer
